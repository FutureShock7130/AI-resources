## About The Project

This project includes guides, skills, and other resources specifically designed for
novice students of FRC and Agnets.



<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

You should have Codex, Cursor, OpenCode, or other  AI coding agents and tools compatible with AGENTS.md,
see [https://agents.md/](https://agents.md/) for all compatible agents.

### Installation

1. Download `AGENTS.md` and save to the root folder of the project
2. Start building something amazing!

## Contributing

Contributions are what make the open source community such an amazing place to learn, inspire, and create. Any contributions you make are **greatly appreciated**.

Don't forget to give the project a star! Thanks again!
